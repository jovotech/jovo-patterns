# Template: Handoff on Twilio Autopilot

Jovo Sample Autopilot Bot that can handoff the session to a human agent using Twilio Flex.

```sh
$ jovo new <directory> --template autopilot/handoff
```

After that you have add your Workflow's ID to the marked section in the handler.
